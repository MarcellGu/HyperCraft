# HyperCraft

HyperCraft 是一个基于 Next.js 和 TypeScript 的极简聊天应用。

## 功能特性
- 多模型选择，支持多种 AI 聊天模型
- 聊天历史管理，支持多会话
- 消息输入与展示，支持流式响应
- 侧边栏导航，便捷切换会话与设置
- 组件化开发，易于维护和扩展
- 支持自定义字体与主题

## 目录结构
- `src/app`：页面与路由
- `src/components`：核心 UI 组件（聊天窗口、历史列表、模型选择等）
- `src/assets/fonts`：内置字体资源
- `src/assets/icons`：图标资源
- `src/hooks`：自定义 React Hooks
- `src/interfaces`：类型定义
- `src/styles`：全局与局部样式
- `src/utils`：工具函数

## 快速开始
1. 安装依赖：
   ```bash
   npm install
   ```
2. 启动开发环境：
   ```bash
   npm run dev
   ```
3. 访问 [http://localhost:3000](http://localhost:3000)

## 依赖环境
- Node.js 18+
- 推荐使用 VSCode 编辑器
